## 🚀 Comment héberger sur GitHub Pages (Gratuit & Vie)

Puisque ce repository est **privé**, suis ces étapes pour activer l'hébergement :

1. Va dans les **Settings** de ce repository.
2. Clique sur **Pages** dans la barre latérale gauche.
3. Sous **Build and deployment** > **Source**, change "Deploy from a branch" par **"GitHub Actions"**.
4. C'est tout ! Le déploiement commencera automatiquement grâce au fichier dans `.github/workflows/deploy.yml`.
